Guia de Aceitação dos Equipamentos

Subir Conexão DWDM (Se houver)

a. Exemplos:
i. Campo Grande - Sidrolândia;
ii. Sidrolândia - Maracaju;
iii. Maracaju - Dourados;
iv. Dourados - Fátima do Sul.

Configurar DWDM

i. Subir NE através de OSC;
ii. Validar níveis de potência em ambos os lados (se estão iguais);
iii. Realizar Conexões Lógicas em Signal Flow;
iv. Realizar Remarks das Placas e Rename das Interfaces/Placas;
v. Criar WDM Trail;
vi. Realizar alinhamento de Canal;
vii. Realizar ajuste de potência/sinal;
viii. Preencher Referência de Potências nas placas AST2, DAP e LDCD;
ix. Validar Backup do Equipamento no NCE.

Subir conexão com Roteador (Se houver):

a. Adicionar Equipamento no Radius: i. Consultar Antônio, Bruno, Fabricio, Hank, Junio ou Paulo para tal procedimento;
ii. Realizar procedimento sempre que subir um novo Equipamento.
b. Validar se os usuários estão autenticando com suas respectivas senhas (Radius);

c. Validar roteamento até o Equipamento (OSPF/MPLS);
i. Precisa checar se há conexão OSPF point-to-point -> display ospf interface (válido somente para enlaces backbone).

d. Validar configurações de ACL, Banner e SNMP Cacti/Zabbix (Válidos para Equipamentos até Três Lagoas; para mais informações consultar Antônio, Paulo ou Fabricio);
e. Adicionar o Equipamento no NCE;
f. Validar conexão do NCE com o Equipamento através de SNMP, STELNET e SNETCONF;
g. Validar Backup do Equipamento no NCE;
h. Organizar Links e Topologia em Topology View no NCE (Fiber Link e L2 Link);
i. Adicionar Equipamento no DNS e validar acesso pelo Reverso;
j. Adicionar Equipamento no CACTI;
k. Adicionar Equipamento no Zabbix;
l. Adicionar Equipamento no Grafana;
m. Preencher a planilha de DADOS DE ENTREGA de acordo com a Pasta do Projeto.

i. Adicionar Equipamento no DNS e validar acesso pelo Reverso;

Configuração de OLT:

a. Adicionar equipamento no Radius:
i. Consultar Antônio, Bruno, Fabricio, Hank, Junio ou Paulo para tal procedimento;
ii. Realizar procedimento sempre que subir um novo Equipamento.

b. Validar se os usuários estão autenticando com suas respectivas senhas (Radius);
c. Validar configurações de ACL (Telnet/SSH) (Válidos para Equipamentos até Três Lagoas; para mais informações consultar Antônio, Paulo ou Fabricio);
d. Adicionar o Equipamento no NCE;
e. Validar conexão do NCE com o Equipamento através de SNMP e STELNET;
f. Validar Backup do Equipamento no NCE;
g. Renomear Interfaces GPON/Ethernet no NCE de acordo com os Ramais (consultar projeto da cidade em questão)
h. Adicionar Equipamento no DNS e validar acesso pelo Reverso;
i. Adicionar Equipamento no Zabbix;
j. Adicionar Equipamento no Grafana;
k. Adicionar Equipamento em Transmissores no IXC e validar conexão IXC com OLT.
l. Organizar ONU Topology View no NCE (de acordo com Projeto do IXC FiberDocs);
m. Preencher a planilha de DADOS DE ENTREGA de acordo com a Pasta do Projeto.
